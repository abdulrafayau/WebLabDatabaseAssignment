## How to Add Screenshots

1. Take screenshots of each page
2. Save as PNG format
3. Name them according to above naming convention
4. Place in this screenshots directory
5. Update README.md to reference the screenshots

## Screenshot Tips

- Use full-width browser window for consistent sizing
- Show populated data (not empty state)
- Include at least one modal/form open
- Use a clean, neutral background
- Ensure text is readable at 1920x1080 resolution
